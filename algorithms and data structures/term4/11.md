## 3

Пусть $a = ud, b = vd$. Решим как решали и получим $(x, y)$. 
$$a(x + kv) + b(y - ku) = ax + by + k(av - bu) = ax + by + k(udv - vdu) = ax + by$$
, т.е. $(x + kv, y - ku)$ - решение $\forall k$.
